# 🤖 AI-Job-Board

## 📌 Overview
AI-Job-Board is an intelligent recruitment platform that connects job seekers with the most relevant job opportunities using AI-driven recommendations. The platform analyzes user skills, resumes, and preferences to provide personalized job suggestions while helping recruiters identify suitable candidates efficiently.

---

## ✨ Features
- 🔍 AI-powered job recommendations
- 📄 Resume upload and analysis
- 🎯 Advanced job search and filtering
- 👤 User authentication and authorization
- 🧑‍💼 Employer and candidate dashboards
- 📌 Job application tracking
- 📱 Responsive and modern UI

## 🚀 Use Case
This platform helps job seekers discover suitable opportunities quickly and efficiently while enabling recruiters to find qualified candidates through intelligent matching and analytics.

## ⚙️ Installation

```bash
# Clone repository
git clone <repository-url>

# Navigate to project folder
cd AI-Job-Board

# Create virtual environment
python -m venv venv

# Activate virtual environment
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run server
python manage.py runserver
```
